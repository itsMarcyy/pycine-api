# Pycine 🎬

API desenvolvida com FastAPI para gerenciamento de mídias e avaliações.

## 🚧 Status
Projeto em desenvolvimento. Ainda estou implementando novas funcionalidades e melhorias.

## 🛠 Tecnologias
- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Pydantic

## ✅ Funcionalidades atuais
- Cadastro de mídias
- Listagem de mídias
- Sistema de avaliações

## 📌 Próximos passos
- Autenticação de usuários
- Sistema de favoritos
- Melhorias na estrutura do projeto
